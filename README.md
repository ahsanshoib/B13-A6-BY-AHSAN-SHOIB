NAME : SUPER-SOFT-SHOP

DESCRIPTION   : 

This is a simple software shopping cart website built with React. Users can see products, add them to the cart, and manage their items easily. The website is responsive and easy to use.

USED TECHNOLOGIES : 

1. React.js
2. Tailwind CSS
3. JavaScript
4. JSON Data

KEY FEATURES :

1. Shopping Cart : Users can add, remove, and update products in the cart easily.
2. Responsive Design : The website works well on mobile, tablet, and desktop devices.
3. Interactive UI : The site is fast and smooth with simple user interactions.
